# Task Flow App 🚀

A responsive Todo app built with React and Tailwind CSS featuring task management and category-based organization.

## Features
- Add / Delete tasks
- Categories (Work, Home, Business)
- Filter tasks
- Responsive UI

## Tech Stack
- React
- Tailwind CSS
- JavaScript

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
